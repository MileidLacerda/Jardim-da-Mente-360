Plataforma web para venda de materiais psicopedagógicos digitais, blog educativo e captação de leads.

---

## 👥 Organização da Equipe

O projeto é desenvolvido de forma colaborativa, com divisões claras de responsabilidades para garantir escalabilidade e segurança.

### 🎨 Frontend
**Integrantes:** Hyuri, Claude, Luni
* **Responsabilidades:** Desenvolvimento de layout, responsividade, páginas públicas, integração com API e otimização da experiência de compra.

### ⚙️ Backend
**Integrantes:** Rafaela, Valdemar, Lizangelo
* **Responsabilidades:** Implementação de autenticação, API REST, gestão de pedidos, processamento de pagamentos, sistemas de downloads e painel administrativo.

### 🗄️ Banco de Dados
**Integrante:** Lizangelo
* **Responsabilidades:** Modelagem de dados, gestão de migrations, criação de índices e manutenção da integridade relacional.

### 🧪 QA (Quality Assurance)
**Integrante:** Mileid
* **Responsabilidades:** Testes de cadastro de produtos, publicação de artigos, fluxo de compra (pagamento aprovado/recusado), liberação de downloads, validação de formulários de leads e testes de interface mobile.

### 🛡️ Análise de Segurança (White-Box Pentester)
**Integrante:** Romulo
* **Responsabilidades:** Auditoria de segurança baseada em análise de código-fonte para identificação de vulnerabilidades (SAST), verificação de conformidade com o OWASP Top 10, sanitização de inputs e mitigação de riscos lógicos no backend e banco de dados.

---

## 🛠️ Tecnologias e Ferramentas

* **Versionamento:** Git & GitHub
* **Segurança:** Static Application Security Testing (SAST)
* **Documentação:** Markdown & Fluxogramas de usuário

---
 
*Este projeto segue diretrizes de Security by Design para garantir a proteção dos dados dos usuários e integridade das transações.*
