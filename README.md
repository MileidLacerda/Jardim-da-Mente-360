# Jardim da Mente 360

Plataforma web para venda de materiais psicopedagógicos digitais, blog educativo, captação de leads e gerenciamento de produtos.

## Objetivo
Desenvolver um sistema próprio para:
- vender materiais psicopedagógicos em PDF
- publicar artigos educativos
- oferecer materiais gratuitos
- captar leads
- gerenciar pedidos e downloads

## Stack
- Frontend: React, HTML, CSS, JavaScript
- Backend: Spring Boot
- Banco de dados: PostgreSQL

## Estrutura do projeto
- `/frontend` → interface do usuário
- `/backend` → API e regras de negócio
- `/database` → scripts, modelagem e migrations
- `/docs` → documentação técnica e funcional

## Fluxo de branches
- `main` → versão estável
- `develop` → integração de desenvolvimento
- `feature/*` → novas funcionalidades
- `fix/*` → correções
- `docs/*` → documentação
